# 🪨 Rock Paper Scissors - Python Tkinter Game

This is a fun GUI-based Rock-Paper-Scissors game built with Python using Tkinter and Pillow libraries.

---

## 🎮 Features

- 🌀 **Animated computer selection** to simulate "thinking"
- 📊 **Leaderboard** showing Wins, Losses, and Ties
- 🎨 Uses images for buttons and results
- 🔄 Reset button to start fresh

---

## 🗂️ Project Structure

```txt

project1/
├── main.py                 # Main game script
├── resources/              # Image assets (rock, paper, scissor)
│   ├── rock.png
│   ├── paper.png
│   └── scissor.png
├── README.md
├── requirements.txt
└── .gitignore

````

---

## ▶️ Getting Started

### 1. Clone this repository

```bash
git clone https://github.com/your-username/project1.git
cd project1
````

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Run the game

```bash
python main.py
```
